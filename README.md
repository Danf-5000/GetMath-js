# GetMath-js
библиотека для упрощения работы с математикой в js
Предположим в нашей библиотеке есть эти функции

Plus(a, b); - сложение

Minus(a, b) - уменьшение

Multiply(a, b) - умножение

Divide(a, b) - деление 

Degree(a, b) - степень

Sin(a) - синус

Cos(a) - косинус

Tan(a) - тангенс

RectangleP(a, b) - периметр прямоугольника

TriangleP(a, b , c) - периметр треугольника

CircleP(r) - периметр круга

a , b можно менять на свои цифры

Пример: Plus(2, 2) - выведет 4

Использования в коде

console.log(Plus(2, 3));

alert(Plus(2, 3));

Пример новых функций

console.log(RectangleP(2, 2));

console.log(TriangleP(2, 2 , 2));

console.log(CircleP(2));
